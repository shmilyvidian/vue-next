### vs-next
> 基于vite的一套pc amdin项目

1. 启动
    ```javascript
    yarn install
    or
    npm i

    npm run dev
    ```

2. 项目特性

   - vue3 [文档](https://v3.vuejs.org/)
   - Typescript [文档](https://www.tslang.cn/docs/home.html)
   - ant-design-vue [文档](https://www.antdv.com/docs/vue/introduce-cn/)
   - vue-router [文档](https://github.com/vuejs/vue-router-next)
   - vuex [文档](https://vuex.vuejs.org/zh/)

3. 项目结构

   ```javascript
   | config  // 配置文件

   | src

       |__ assets      // 静态资源

       |__ components  // 组件

       |__ utils     // 工具方法

       |__ store      // 状态

       |__ router      // 路由

       |__ style		// 样式

       |__ views		// 页面视图

   ```
4. 打包部署
    ```javascript
    npm run build
    ```
